
Process for running app:

1. Unzip the project and open klorofil(project name) folder in command prompt
2. In klorofil folder, all required python libraries are listed in requirements file.
3. Install those python libraries using below command.
		pip install -r requirements.txt
4. Run the django application by using below command. This command should be run from klorofil folder only.
		python manage.py runserver 0.0.0.0:80
5. Now, we can access application with http://localhost
6. Click on Calendar Events link from left side bar.
7. From this page we can add, update & delete our calendar events.
